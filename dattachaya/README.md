# Temple Website

A simple static website structure for a temple, including image slider, gallery, YouTube embed, and Google Map.

## Structure
- `index.html`: Main entry point
- `assets/images/slider/`: Slider images
- `assets/images/gallery/`: Gallery images
- `assets/images/logo/`: Logo images
- `css/`: Stylesheets (base, layout, components, responsive)
- `js/`: JavaScript files (main, slider, gallery, YouTube, map)
- `data/`: JSON data for slider, gallery, videos, and temple info

## Usage
Add your images and update the JSON files as needed.
